# Yahoo.comのアカウントを削除した話

2016年12月15日付で、Yahoo.comのアカウントを削除した。

# Yahoo.comはほとんど使っていなかった

アメリカのYahoo.comとは別のサービスとして日本にはヤフージャパンがあるので、日本人として日本から利用するにあたってYahoo.comのサービスはほとんど必要なかった。だからもともと使っていなかったのだが、唯一Flickrだけがヤフージャパンにはないサービスで日本からも利用者が多いサービスだった。

# アカウント情報漏洩はひとつのきっかけ

そのような状況でYahoo.comのアカウント情報漏洩のニュースが入ってきた。パスワードはサイトごとに変えているし、Yahoo.comはアカウント名すら通常使っているものとは別立てだったのでそれほど心配する必要はなかったのだが、もう使っていないのだからということでこれをきっかけにアカウントを削除した。

削除するにあたっては前述のFlickrだけが気がかりだった。今やFlickrには投稿していないとはいえ、以前活発に活動していた頃のつながりが残っている。とはいえ、ほとんどの人はFlickr以外でも連絡を取ることはできる。ここは思い切りが必要と、アカウントを削除した。

# Flickrのアカウントが宙吊りに

しかしここで問題が発生した。Flickrのアカウントは削除されないまま、ログインにはYahoo.comのアカウントが必要なのでログインすらできない幽霊アカウントになってしまった。ものは試しとYahoo.comにもう一度ログインしてみるとあっさりアカウントが復活したので、先にFlickrのアカウントを削除してからYahoo.comのアカウントを再度削除した。とはいえ、Yahoo.comは本当に削除できているのだろうか。

通知メールによるとデータは91日間保持されてから削除されるそうなので、それまでは再度ログインすれば復活するのだろうか。そのあたりの説明が不十分なあたり管理がずさんな感じがする。

# 新たな展開で更に不安が増大

そうこうしているうちに、Yahoo.comは事業をComcastに売り渡して自身は社名を変えると発表した。まだ削除されていないアカウントデータ、ちゃんと扱ってもらえるのだろうか。ちょっと心配だ。
